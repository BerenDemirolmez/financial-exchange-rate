About These Charts

These charts focus on debt-based FER for euro-area countries using the currently available other-debt series. The plotted values are quarterly indices, with assets, liabilities, and net positions available through the metric selector.

The current website view uses the aggregate BIS LBS other-debt construction. Later versions can extend this layout with sector-specific views while keeping the same comparison interface.

Method

The current construction is based on aggregated BIS LBS other-debt FER series for euro-area countries. This page is designed so that sector-level FER views can be added later without changing the overall navigation or chart workflow.
